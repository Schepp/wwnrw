--- datum: "08.08.2013" talks: - title: Einführung in Require.js author: Frederic Hemberger twitter: "@fhemberger" description: | Websites werden immer komplexer und beinhalten oftmals viel JavaScript. Komplexer Code erleichtert nicht gerade die Übersicht und die Fehlersuche dauert oftmals deutlich länger. Wird zudem viel Code beim Seitenaufruf ausgeführt, kann das auch negative Auswirkungen auf die Performance haben. Require.js bietet die Möglichkeit, JavaScript in kleinen, übersichtlichen Modulen zu verwalten und Code-Abhängigkeiten zu Bibliotheken und Plug-ins zu definieren.
