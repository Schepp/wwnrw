Text: Code wird öfter gelesen als geschrieben. 80 % der Lebensdauer einer Software entfallen im Schnitt auf den Wartungszeitraum. Leider ist es im Normalfall mühsam sich in Code einzulesen - sogar in eigenen, wenn er nur genügend alt ist. (Also ca. 2 Wochen alt.)

Die Clean Code Bewegung möchte das verbessern, damit wir alle in Zukunft leicht lesbaren, wartbaren und erweiterbaren Code erzeugen bzw. vorfinden. Corinna stellt Clean Code im Allgemeinen, sowie ein paar konkrete Methoden am praktischen Beispiel vor.
